# ClawDoctor 🦞

OpenClaw Health Monitor & Fixer

OpenClaw 健康监控与修复工具

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 功能特性 | Features

- 🔍 **实时监控** - 监控 Gateway、技能、系统资源
- 🔧 **一键修复** - 自动修复常见问题
- 🛡️ **安全扫描** - 检测安全风险
- 📊 **数据可视化** - 图表展示资源趋势
- 🌐 **中英文切换** - 支持中英文界面

## 安装 | Installation

```bash
# 克隆仓库
git clone https://github.com/yourusername/clawdoctor.git
cd clawdoctor

# 安装依赖
pip3 install psutil

# 启动服务
python3 server_simple.py
```

## 使用 | Usage

```bash
# 查看状态
python3 clawdoctor_simple.py --status

# 一键修复
python3 clawdoctor_simple.py --fix

# 安全扫描
python3 clawdoctor_simple.py --scan
```
